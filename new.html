üéØ Introduction

WiFi networks face growing threats from sophisticated cyberattacks, necessitating advanced intrusion detection systems. This research presents a comprehensive comparison of 28 machine learning algorithms across three paradigms using the AWID dataset containing 1,795,575 network traffic samples with 154 features.

üöÄ Goal

Identify the most effective machine learning approach for WiFi intrusion detection by comparing:

  • Classical ML: Ensemble methods, gradient boosting
  • Traditional NN: Multi-layer perceptrons
  • Deep Learning: CNN, LSTM, Transformer models

üí° Motivation

  • Detect sophisticated attack vectors (injection, impersonation, flooding)
  • Provide real-time, high-accuracy intrusion detection
  • Enable evidence-based algorithm selection for deployment
  • Balance accuracy with computational efficiency

üìö Related Work

  • Limited comparative studies across ML paradigms for WiFi IDS
  • Ensemble methods show promise but lack systematic comparison
  • Deep learning applications focus on individual architectures
  • Gap in practical deployment considerations

üî¨ Methodology

Dataset Preprocessing:

  • Missing value analysis and feature reduction (72 features removed)
  • SMOTE-based class balancing for imbalanced data
  • Final dataset: 1,339,406 instances with 73 features

Model Categories:

  • Classical ML: Random Forest, XGBoost, LightGBM, Bagging
  • Traditional NN: Various MLP architectures
  • Deep Learning: DNN, CNN-1D, LSTM, GRU, Transformer

üéØ Conclusion

We developed a comprehensive evaluation framework comparing 28 machine learning algorithms for WiFi intrusion detection. Classical ensemble methods significantly outperformed neural approaches, with LightGBM achieving the optimal balance of 99.91% accuracy and 10.72s training time.
